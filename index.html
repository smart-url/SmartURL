Integration Guide (API)

How to Shorten

Send a POST request with a JSON body containing your target URL.

POST /smarturl-api
{
  "url": "https://example.com"
}

The Response

You will receive a unique short code back as a JSON object.

{
  "shortUrl": "a7b9x2"
}
